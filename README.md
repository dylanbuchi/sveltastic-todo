# sveltastic-todo

# ⌨️ Commands

All commands are run from the root of the project, from a terminal:

| Command        | Action                                      |
| :------------- | :------------------------------------------ |
| `pnpm install` | Installs dependencies                       |
| `pnpm dev`     | Starts local dev server at `localhost:5173` |
| `pnpm build`   | Build your production site                  |
